# Customer_Behavior_Analysis
Data Analysis and Interactive Dashboard project built using Excel/Power BI to analyze sales performance, customer trends, and business insights.


📊 Data Analytics Project – End-to-End Workflow
🔎 Overview

This project demonstrates a complete, industry-standard data analytics workflow. It covers the full data lifecycle — from raw data loading and cleaning to SQL analysis and interactive dashboard creation.

The objective of this project is to extract meaningful insights from data and present them in a clear, business-friendly format using modern analytics tools.

This project reflects real-world responsibilities of a Data Analyst, including:

Data preparation

Exploratory Data Analysis (EDA)

SQL-based querying

Data visualization

Insight generation and reporting

📁 Dataset

The dataset used in this project contains structured business data (e.g., sales, customers, or transactions).

It includes:

Numerical features (e.g., revenue, quantity, price)

Categorical features (e.g., product category, region)

Date/time fields

The dataset was cleaned and transformed before analysis to ensure accuracy and reliability of insights.

⚙️ Project Steps
1️⃣ Data Loading (Python)

Imported dataset using pandas

Checked structure, data types, and missing values

2️⃣ Data Cleaning

Handled missing values

Removed duplicates

Corrected inconsistent formats

Converted data types where necessary

3️⃣ Exploratory Data Analysis (EDA)

Summary statistics

Distribution analysis

Correlation analysis

Trend analysis

Identified key patterns and anomalies

4️⃣ SQL Analysis (PostgreSQL)

Imported cleaned data into PostgreSQL

Wrote SQL queries to calculate:

Total revenue

Sales by region

Top-performing products

Monthly trends

Used GROUP BY, JOIN, ORDER BY, and aggregate functions

5️⃣ Power BI Dashboard

Connected PostgreSQL to Power BI

Created data model and relationships

Built calculated measures using DAX

Designed interactive dashboard with filters and slicers

📊 Dashboard

The Power BI dashboard includes:

KPI cards (Total Revenue, Total Orders, etc.)

Sales trend over time

Region-wise performance

Top products/customers

Interactive filters for dynamic analysis

The dashboard enables stakeholders to quickly understand performance and make data-driven decisions.
